# 🤝 CONTRIBUTING — NEXUS ESPORTS

## 🛑 FEATURE FREEZE

**No new features until launch.**

Allowed:
- ✅ Bug fixes
- ✅ Refactors
- ✅ Performance improvements
- ✅ Documentation updates
- ✅ Database optimization
- ✅ Error handling

Not Allowed:
- ❌ New pages
- ❌ New API routes
- ❌ New database models
- ❌ New features
- ❌ Major architectural changes

---

## 📋 Development Process

### 1. Before Starting
- Check `TODO.md` for what's next
- Check `BUGS.md` for what to fix
- Check `ROADMAP.md` for phase progress

### 2. During Development
- Follow the phase order (Phase 1 → 2 → 3 ...)
- One phase at a time
- Test after each change
- Update docs after completion

### 3. Before Committing
- `npm run build` (must pass)
- Test locally
- Update `ROADMAP.md` if needed
- Update `BUGS.md` if fixed
